#  Medicinalis - Sistema de Recomendaciones Nutricionales Basado en Análisis Clínicos
**Medicinalis** es una base de datos diseñada para organizar estudios clínicos, biomarcadores y condiciones nutricionales, generando **recomendaciones automáticas** basadas en análisis de sangre.

## Objetivo
Centralizar la información de pacientes y resultados, automatizar la detección de valores fuera de rango y facilitar recomendaciones nutricionales confiables.

##  Componentes del sistema
- **Modelo DER**.  
- **Tablas principales:** Paciente, Estudio, Resultado, Biomarcador, Condición, Recomendación.  
- **Tablas de implementación:** hechos, transacciones y categorías.  
- **Vistas:** historial clínico, biomarcadores fuera de rango, recomendaciones, resumen de estudios.   
- **Stored Procedures:** carga de estudios y generación automática de condiciones.  
- **Triggers:** análisis automático al insertar o actualizar resultados.

## Tecnologías utilizadas
MySQL · SQL · Lucidchart · Power BI · LaTeX · VSCode
