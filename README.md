# Medicinalis

Medicinalis es una base de datos relacional diseñada para registrar estudios clínicos, interpretar resultados y generar recomendaciones nutricionales basadas en dichos valores.  
El sistema automatiza la evaluación de resultados y asegura la trazabilidad desde el paciente hasta la recomendación final.


### Funcionamiento
- Registrar pacientes, estudios, biomarcadores y resultados.
- Evaluar automáticamente si un resultado está dentro o fuera del rango de referencia.
- Asociar resultados con condiciones clínicas y recomendaciones personalizadas.
